Networks 1 - Glossary

Glossary NW1
Network (Graph): A mathematical structure used to represent relationships between entities, consisting of nodes (vertices) and edges (links).
Node (Vertex): An individual entity within a network, such as a gene, protein or station in a transport system.
Edge (Link / Arc): A connection between two nodes, representing a relationship or interaction.
Undirected Network: A network in which edges have no direction, implying a mutual or symmetric relationship.
Directed Network (DiGraph): A network in which edges have direction, indicating flow from a source node to a target node.
Weighted Edge: An edge associated with a numerical value representing strength, reliability or magnitude of interaction.
Bipartite Network: A network in which nodes are divided into two distinct sets and edges only connect nodes from different sets.
Adjacency Matrix: A square matrix representation of a network where rows and columns correspond to nodes and matrix entries indicate the presence or weight of edges.
Edge List: A tabular representation of a network where each row defines a connection between two nodes, optionally including an edge weight.
Graph Object (NetworkX Graph): A Python data structure created using NetworkX that stores nodes, edges and their attributes for analysis.
Degree: The number of edges connected to a node in an undirected network.
In-degree: The number of incoming edges to a node in a directed network.
Out-degree: The number of outgoing edges from a node in a directed network.
Degree Distribution: The frequency distribution of node degrees across a network.
Degree Centrality: A normalised measure of how many connections a node has relative to the maximum possible number of connections.
Closeness Centrality: A measure of how close a node is to all other nodes in a network, based on shortest path distances.
Shortest Path: The minimum number of edges required to travel between two nodes.
Average Shortest Path Length: The mean shortest path distance between all pairs of nodes in a network.
Centrality Score: A quantitative measure describing the structural importance or influence of a node within a network.
PageRank: A probabilistic ranking algorithm based on random walks and teleportation, used to estimate node importance.
Personalised PageRank (PPR): A variant of PageRank that biases the random walk towards a specific node or set of nodes.
Teleportation (PageRank Context): The random jump to any node in the network, preventing the random walk from becoming trapped.
Subgraph: A smaller network extracted from a larger graph, containing a subset of nodes and edges.
Layout Algorithm: A method used to determine the spatial arrangement of nodes when visualising a network (e.g. spring layout, spiral layout, spectral layout).
Spring Layout: A force-directed layout algorithm that positions nodes based on simulated attractive and repulsive forces.
Spectral Layout: A layout method based on eigenvectors of a matrix representation of the graph.
Clustering Coefficient: A measure of the extent to which a node’s neighbours are connected to each other.
Average Clustering Coefficient: The mean clustering coefficient across all nodes in a network.
Edge Connectivity: The minimum number of edges that must be removed to disconnect a network.
Connected Component: A subset of nodes within a network where every node is reachable from every other node in the same subset.
Protein–Protein Interaction (PPI) Network: A biological network where nodes represent proteins and edges represent physical or functional interactions.
Gene Regulatory Network (GRN): A directed network describing regulatory interactions between genes and transcription factors.
Network Visualisation: The graphical representation of a network structure to support interpretation of connectivity and node importance.
