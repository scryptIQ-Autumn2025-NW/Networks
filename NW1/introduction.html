    Introduction NW1

    Welcome to the Networks module of the scryptIQ course. Networks provide a powerful mathematical framework for representing complex systems composed of interacting components. In the biosciences, they are often commonly used to model relationships between genes, proteins, metabolites, diseases and even individuals in populations. By representing biological entities as nodes and their relationships as edges, networks allow us to move beyond studying isolated components and instead examine how structure, connectivity and organisation might elucidate valuable information about biological function.

    In this lesson, we introduce the fundamental concepts of network graph theory and explore how to construct, analyse and visualise networks using Python and the NetworkX package. We will explore how different network representations capture distinct types of biological relationships, how to quantify structural properties using centrality measures and how ranking algorithms such as PageRank can help prioritise influential nodes. By combining quantitative metrics with visualisation, we will develop the tools needed to extract meaningful biological insight from real-world network data.

    Concept Synergy Syntax & Functions Objective / Application Common Pitfalls / Notes
    Network Concepts (Nodes & Edges) Extends earlier data structure thinking (arrays, matrices) to relational systems Graph terminology: nodes, edges, attributes Understand how biological entities and their relationships are represented mathematically Confusing nodes with edges; assuming all networks imply direction or weight
    Network Types (Directed, Undirected, Weighted, Bipartite) Connects biological examples (PPI, GRN, disease networks) to graph structure nx.Graph(), nx.DiGraph() Distinguish structural differences and interpret biological meaning correctly Forgetting to specify directed graphs; misinterpreting weights as direction
    Network Representation (Adjacency Matrix vs Edge List) Builds on NumPy arrays and Pandas DataFrames nx.from_numpy_array(), nx.from_edgelist(), nx.from_pandas_edgelist() Convert structured data into NetworkX graph objects efficiently Using dense adjacency matrices for sparse large networks; mixing up row/column meaning
    Accessing and Relabelling Nodes Reinforces dictionary and mapping concepts from earlier Python lessons G.nodes(), G.edges(), nx.relabel_nodes() Manage and interpret node labels clearly in analysis Losing original labels when creating graphs from arrays
    Degree & Degree Centrality Extends earlier counting and normalisation concepts G.degree(), nx.degree_centrality() Identify highly connected or influential nodes Confusing raw degree with normalised centrality
    Closeness Centrality Builds on shortest path intuition nx.closeness_centrality() Identify nodes that efficiently spread information through the network Interpreting closeness without considering disconnected components
    Personalised PageRank (PPR) Connects probabilistic thinking to network influence nx.pagerank(personalization=...) Prioritise nodes relative to a biologically important seed (e.g. EGFR) Forgetting to define the personalisation dictionary correctly; misinterpreting scores as absolute importance
    Network Visualisation & Layouts Extends matplotlib plotting from earlier lessons nx.draw(), spring_layout(), shell_layout(), random_layout() Interpret structural patterns visually and communicate results clearly Assuming layout reflects true biological geometry; misreading visual density as importance
    Clustering & Connectivity (Optional) Extends graph metrics to structural robustness nx.clustering(), nx.edge_connectivity() Assess modularity and robustness of biological networks Confusing clustering coefficient with centrality; misinterpreting disconnected components
