    Introduction NW2

    Welcome to the second lesson in scryptIQ's networks module. In this lesson, we will extend the fundamentals of network construction to the application of network analysis in real biological systems. You will learn how to import network data from multiple formats such as CSV files, edge lists, and JSON files - all of which are common formats for storing data from which to build network graphs. We will learn how to take these source files and convert them into usable NetworkX graph objects and, along the way, we will reintroduce the key structural concepts including nodes, edges, weighted networks, bipartite graphs, and the importance of layout selection for meaningful visualisation, as an extension of Networks 1.

    This lesson will then build upon these foundations by generating random networks to test hypotheses and understand the role of reproducibility using random seeds.

    Finally, we will also learn how to apply core analytical tools such as degree centrality, closeness centrality, Personalised PageRank (PPR) and shortest path analysis to a real-world protein–protein interaction network derived from the STRING database, and focusing on key targets in the Breast Cancer Network and Pancreatic Cancer Network. Through this progression, the lesson aims to emphasise the important principles of network science: where biological insight emerges not only from identifying connections, but from quantifying and interpreting the structure and flow of information within a network. These lessons begin to give an insight into the analytical and informative depth that can emerge from representing connectedness and relationships as network graphs, and aims to open your mind to potentially applying these approaches to your own research.

    Concept Synergy Syntax & Functions Objective / Application Common Pitfalls / Notes
    Importing Network Data (Adjacency Matrices, Edge Lists, JSON) Builds directly on Pandas, NumPy arrays, and dictionary structures from earlier lessons pd.read_csv(), nx.from_numpy_array(), nx.read_edgelist(), nx.json_graph.node_link_graph() Convert structured biological or text-based datasets into usable NetworkX graph objects Mixing up row/column meaning in adjacency matrices; forgetting to convert DataFrame to NumPy array; losing labels during conversion
    Graph Construction & Basic Properties Reinforces object-oriented thinking in Python nx.Graph(), G.number_of_nodes(), G.number_of_edges(), nx.is_directed(), nx.is_weighted() Understand how biological systems are represented mathematically as nodes and edges Assuming networks are directed or weighted by default; not checking graph type explicitly
    Random Network Generation & Reproducibility Connects probability and random number generation to structural modelling randint(), seed(), nx.from_numpy_array() Generate test networks to explore structural properties and null models Forgetting to set a seed for reproducibility; misinterpreting random structure as biologically meaningful
    Network Layout & Visualisation Extends matplotlib plotting from earlier lessons into relational data nx.draw(), nx.random_layout(), nx.spring_layout(), nx.bipartite_layout() Visually interpret structure, connectivity, and modularity Assuming layout reflects true biological geometry; confusing visual density with importance
    Bipartite Networks Reinforces attribute handling and node grouping concepts add_nodes_from(..., bipartite=0/1), nx.is_bipartite(), nx.bipartite_layout() Represent relationships between two distinct biological entity types Forgetting to define bipartite attributes; confusing connectivity with bipartite structure
    Centrality Measures (Degree & Closeness) Extends counting and shortest-path intuition into quantitative influence metrics G.degree(), nx.degree_centrality(), nx.closeness_centrality() Identify structurally important proteins within interaction networks Confusing raw degree with normalised centrality; ignoring disconnected components when interpreting closeness
    Personalised PageRank (PPR) Connects probabilistic modelling to biological prioritisation nx.pagerank(personalization=...) Quantify influence relative to a biologically meaningful seed node (e.g., TP53) Forgetting to define personalisation dictionary correctly; interpreting scores as absolute rather than relative importance
    Shortest Path & Path Weight Analysis Builds on graph traversal and edge-weight concepts nx.all_shortest_paths(), nx.path_weight() Identify minimal interaction routes between key cancer genes (e.g., KRAS–TP53) and compare interaction strength Misinterpreting weight as distance; failing to normalise by path length when comparing networks
    Subgraph Extraction from Large Networks Connects filtering in Pandas to structural subsetting in NetworkX G.subgraph(node_list) Construct disease-specific networks from a global PPI network (e.g., STRING) Forgetting that subgraph views are tied to the original graph unless copied
    Network Perturbation & Comparative Analysis Extends hypothesis testing to structural interventions G.copy(), G.remove_node() Evaluate how removing a key protein alters signal propagation and shortest paths Forgetting to copy graph before modification; comparing results without controlling for structural changes
