Networks 2 - Glossary

Glossary NW2
NetworkX: A Python library for creating, manipulating and analysing complex networks and graph structures.
Graph Object: The core NetworkX data structure representing a network, containing nodes, edges and optional attributes.
Subgraph: A smaller network extracted from a larger graph, containing a subset of nodes and the edges between them.
Layout: An algorithm that determines how nodes are positioned for visualisation (e.g. spring, random or bipartite layout).
Spring Layout: A force-directed layout algorithm that positions nodes based on simulated physical forces to reveal structural patterns.
Random Layout: A layout that assigns node positions randomly, often used for quick visual inspection.
Bipartite Attribute: A node attribute used to distinguish membership of one of two disjoint node sets in a bipartite network.
Random Network: A network generated using probabilistic rules rather than empirical biological data.
Seed (Random Seed): A fixed initial value used in random number generation to ensure reproducible results.
Reproducibility: The principle that computational results should be repeatable when the same inputs and seed values are used.
Degree Centrality: A normalised measure of how many connections a node has relative to the size of the network.
Closeness Centrality: A measure of how efficiently a node can reach all other nodes via shortest paths.
Personalised PageRank (PPR): A variant of PageRank that biases influence scores towards a specified seed node.
Personalisation Dictionary: A dictionary defining which node(s) receive initial probability mass in a Personalised PageRank calculation.
Shortest Path: The minimum-length sequence of edges connecting two nodes in a network.
Path Weight: The sum of edge weights along a path, representing cumulative interaction strength or confidence.
Path Length Normalisation: Dividing total path weight by the number of edges in the path to allow fair comparison between paths of different lengths.
Protein–Protein Interaction (PPI) Network: A biological network where nodes represent proteins and edges represent physical or functional interactions.
STRING Database: A curated database of known and predicted protein–protein interactions, commonly used to construct PPI networks.
PageRank Score: A probabilistic measure of node importance based on network connectivity structure.
Network Perturbation: The modification of a network (e.g. removing a node) to study its structural or functional impact.
Connectivity: A property describing whether nodes in a network are reachable from one another.
Weighted Network: A network in which edges carry numerical values representing strength, confidence or magnitude of interaction.
